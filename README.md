# alexaa

 Alexa Voice Assistant (Python)

A Python-based Voice Assistant that listens to your voice, executes commands, plays YouTube songs, fetches Wikipedia info, tells jokes, and interacts like a mini-AI assistant.
This project uses speech recognition, text-to-speech, YouTube automation, and Wikipedia search.

 Features

 Voice Recognition (Google Speech API)

 Natural text-to-speech replies

 Play any song on YouTube

 Tell current time

 Wikipedia one-line summary

 Tell random jokes
